Backup

  • Backup

If you need to format your computer or install Vemto on another computer, you can backup the vemto/storage folder to move your projects. By default, you can find this folder under the following directories:

  • on Linux: ~/.config/vemto/storage
  • on macOS: ~/Library/Application Support/vemto/storage
  • on Windows: %USERPROFILE%\AppData\Roaming\vemto\storage
